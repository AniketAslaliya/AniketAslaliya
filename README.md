### 🔬 DebateFloor — deep dive

> *Built in 48 hours. Placed 2nd at the Meta × PyTorch OpenEnv Hackathon Grand Finale, April 25–26 2026.*

Most RL environments reward **what** an agent decides.  
DebateFloor rewards **how confidently** it decides — and whether that confidence was warranted.

Before every terminal action, the agent must declare `HIGH / MED / LOW` confidence. Wrong + overconfident = worst possible outcome. The only winning strategy: accurate calibration.

```
INVESTIGATOR  →  validates docs, flags fraud signals, queries history
      ↓
convene_debate_panel()
      ↓
PROSECUTOR ⚔️  vs  DEFENDER 🛡️   (adversarial reasoning, independent contexts)
      ↓
JUDGE decides: approve / deny / escalate  +  confidence declaration
      ↓
Reward via 3×2 calibration matrix
```

After GRPO training: HIGH confidence dropped from ~82% → ~44%. The model learned *when* to be confident — not just what to say.

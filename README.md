### рЯФђ DebateFloor вАФ deep dive

> *Built in 48 hours. Placed 2nd at the Meta √Ч PyTorch OpenEnv Hackathon Grand Finale, April 25вАУ26 2026.*

Most RL environments reward **what** an agent decides.  
DebateFloor rewards **how confidently** it decides вАФ and whether that confidence was warranted.

Before every terminal action, the agent must declare `HIGH / MED / LOW` confidence. Wrong + overconfident = worst possible outcome. The only winning strategy: accurate calibration.

```
INVESTIGATOR  вЖТ  validates docs, flags fraud signals, queries history
      вЖУ
convene_debate_panel()
      вЖУ
PROSECUTOR вЪФпЄП  vs  DEFENDER рЯЫ°пЄП   (adversarial reasoning, independent contexts)
      вЖУ
JUDGE decides: approve / deny / escalate  +  confidence declaration
      вЖУ
Reward via 3√Ч2 calibration matrix
```

After GRPO training: HIGH confidence dropped from ~82% вЖТ ~44%. The model learned *when* to be confident вАФ not just what to say.
